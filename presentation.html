MyShop

An E-Commerce Platform Built with Node.js & Express

A modern web application for product browsing and user authentication

Project Overview

MyShop is a full-stack e-commerce web application designed to demonstrate:

  • User authentication and login functionality
  • Product listing and product detail pages
  • Client-side search and filtering
  • Responsive UI/UX design
  • Database integration with MongoDB

Target Users: Customers who want to browse and search for products online

Technology Stack

Backend
Node.js + Express.js
REST API endpoints
Database
MongoDB
Mongoose ODM
Frontend
HTML5, CSS3, JavaScript
EJS Templating
Authentication
Session-based login
Email/Password validation

Application Architecture

Backend Structure:
  • app.js - Main Express server & route handlers
  • user.js - User schema and model definition
  • product.js - Product schema and model definition
Frontend Structure:
  • views/ - EJS template files (login, products, product detail)
  • public/ - Static assets (images, favicon, logo)
  • Client-side JavaScript for interactivity & form handling
